# Hardware Troubleshooting Guide

## Issue: Printer Not Working

### Symptoms
- Printer does not respond when printing.
- Error message displayed on printer screen.
- Print jobs stuck in queue.

### Possible Causes
- Printer is turned off or disconnected.
- Paper jam inside the printer.
- Printer driver is outdated or corrupted.
- Network connectivity issue (for network printers).

### Step-by-Step Solution
1. **Check Power and Connections**
   - Ensure the printer is plugged in and powered on.
   - Verify USB or network cable is securely connected.

2. **Clear Paper Jams**
   - Open the printer cover and remove any jammed paper.
   - Check the paper tray for proper alignment.

3. **Restart Printer and Computer**
   - Turn off the printer and computer.
   - Wait 30 seconds, then turn them back on.

4. **Reinstall Printer Driver**
   - Go to the manufacturer’s website.
   - Download and install the latest driver for your printer model.

5. **Check Network Settings**
   - For network printers, ensure the printer is connected to the correct Wi-Fi or Ethernet network.
   - Ping the printer’s IP address to confirm connectivity.
